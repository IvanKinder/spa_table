# backend

## Project setup
```
pip install -r requirements.txt
```

### Run backend server
```
cd backend

python manage.py runserver
```

# frontend

## Project setup
```
cd frontend

npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
